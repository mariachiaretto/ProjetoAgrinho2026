# 🌱 Projeto Agrinho 2026 - Agricultura Familiar

## 📌 Sobre o Projeto
Este é um site desenvolvido para o **Programa Agrinho 2026** (SENAR/PR). O objetivo da página é informar e conscientizar o público sobre a importância da **Agricultura Familiar**, destacando sua relevância na produção de alimentos, na economia e no uso de práticas sustentáveis, com um foco especial no estado do Paraná.

## 🚀 Funcionalidades
* **Design Responsivo:** O site se adapta a diferentes tamanhos de tela (computadores, tablets e celulares).
* **Navegação Intuitiva:** Menu fixo no topo para facilitar o acesso às diferentes seções do conteúdo.
* **Cards Interativos:** Componentes visuais que reagem ao passar o mouse (efeito *hover*) e ao clique, utilizando JavaScript para exibir mensagens dinâmicas.
* **Conteúdo Educativo:** Informações claras e diretas sobre o que é a agricultura familiar, dados de produção (mandioca, leite, café) e inovações no campo.

## 🛠️ Tecnologias Utilizadas
Este projeto foi construído utilizando as bases fundamentais do desenvolvimento web (Front-end):

* **HTML5:** Para a estruturação semântica e acessível do conteúdo (uso de tags como `<header>`, `<main>`, `<section>`, `<footer>`).
* **CSS3:** Para a estilização, cores (paleta baseada na natureza), tipografia (Google Fonts) e responsividade (Media Queries).
* **JavaScript:** Para adicionar interatividade, como o sombreamento dinâmico do cabeçalho ao rolar a página e os alertas nos cards de produtos.

## ⚙️ Como executar o projeto
1. Faça o download ou clone este repositório para o seu computador.
2. Abra a pasta do projeto.
3. Dê um duplo clique no arquivo `index.html` para abri-lo diretamente no seu navegador padrão (Chrome, Edge, Firefox, etc.).
4. *Opcional:* Se estiver usando o VSCode, você pode instalar a extensão **Live Server** e clicar em "Go Live" para visualizar o site com atualização em tempo real.
